# Stock Keymaster 🔑

*Генеруй метадані для стокових фото та відео за допомогою ШІ без меж.*

**Stock Keymaster** — це швидкий вебдодаток, що працює виключно на стороні клієнта. Він використовує Google Gemini API для автоматичної генерації комерційно привабливих назв, детальних описів та релевантних ключових слів для стокових матеріалів (фотографій та відео). Ідеально підходить для контриб'юторів Adobe Stock.

## ✨ Основні можливості

- **Без бекенду:** Працює повністю у вашому браузері. Ваші файли нікуди не завантажуються і залишаються на вашому ПК.
- **Підтримка фото та відео:** Автоматично витягує кадри з відео (на 10%, 50% та 90% тривалості) для точного аналізу суті ролика.
- **Розумна генерація:** Створює заголовок на 5-10 слів, опис з 2-3 речень та рівно 49 релевантних ключових слів.
- **Прев'ю при наведенні:** Миттєво переглядайте фото або програвайте відео-прев'ю, просто навівши курсор на мініатюру.
- **Двомовний інтерфейс:** Легке перемикання між англійською та українською мовами (збереження вибору в cookies).
- **Копіювання в один клік:** Зручне копіювання згенерованих даних у буфер обміну.
- **Відмовостійкість:** Вбудована обробка помилок та автоматичне відновлення після лімітів API (автоматичні паузи).

## 🚀 Як використовувати

1. Склонуйте цей репозиторій або просто завантажте файли.
2. Відкрийте файл `index.html` у будь-якому сучасному браузері.
3. Отримайте безкоштовний [Google Gemini API ключ](https://aistudio.google.com/app/apikey).
4. Вставте ключ у додаток та натисніть **Підключити**.
5. Натисніть **Вибрати папку** та вкажіть локальну теку з вашими медіафайлами.
6. Натисніть **Почати генерацію** та копіюйте готові результати!
